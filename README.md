# Helpa — исследование бренда кредитного мобильного приложения

## О проекте

Helpa — маркетинговое исследование бренда кредитного мобильного приложения перед запуском на индийском рынке. Проект включает тестирование восприятия бренда на реальной целевой аудитории: заёмщиках из городов Tier-1 и Tier-2 Индии.

## Цель

Оценить, как целевая аудитория воспринимает бренд Helpa — название, интерфейс приложения, страницу в Google Play — и выявить барьеры, страхи и точки роста. Результат исследования — обоснованный вердикт: запускать, доработать или отказаться.

## Методология

Исследование сочетает качественные и количественные методы:

- **Глубинные интервью (IDI)** — индивидуальные онлайн-сессии с верифицированными заёмщиками для детальной проработки тем доверия, страхов и негативного опыта
- **Фокус-группы** — групповые онлайн-сессии для обсуждения ассоциаций с брендом и сравнения с конкурентами (KreditBee, MoneyTap, CASHe и др.)
- **Онлайн-анкетирование** — количественная часть для статистической валидации качественных инсайтов

Выборка — 50+ респондентов. Многоступенчатая верификация: скрининговая анкета, подтверждение опыта использования кредитных приложений, при необходимости — видеоверификация.

## Deliverables

- Аналитический отчёт с выводами и рекомендациями
- Карта восприятия бренда (визуализация ассоциаций и эмоций)
- Вербатимы — ключевые цитаты респондентов
- Количественные срезы по результатам анкетирования
- Видеозаписи интервью и фокус-групп
- Финальный вердикт с рекомендациями по каждому сценарию

## Статус

Проект на этапе подготовки — разработка методологии, инструментов и запуск рекрутинга респондентов.
