## 1. Суть проекта

Международная финтех-компания выходит в Индию с кредитным приложением **«Helpa»**. Цель — протестировать бренд на реальных индийских заёмщиках и выдать вердикт: **запускать / доработать / отказаться**.

**Что тестируем:**
1. Восприятие названия — ассоциации, доверие, запоминаемость
2. Прототипы приложения — удобство, понятность, безопасность
3. Страница Google Play — что привлекает, что отталкивает, что побуждает установить
4. Барьеры и страхи цифрового кредитования
5. Сравнение с конкурентами — KreditBee, MoneyTap, CASHe и др.

## 3. Методология

Клиент предпочитает фокус-группы. Предлагаем **три варианта** + параллельное онлайн-анкетирование.

> **Клиент выбирает между Вариантами B и C.** Возможен промежуточный вариант.

### Почему 50 респондентов — минимум

Бриф просит 15–24 человека, но для статистически значимых выводов нужно минимум 50. В Индии набрать такое количество реально (подтверждено Дипаншу).

### Структура: кач + кол

| Компонент | Формат | Количество | Цель |
| :--- | :--- | :--- | :--- |
| **Качественная часть** | IDI + фокус-группы | 18–24 респ. | Глубина: доверие, страхи, ассоциации |
| **Количественная часть** | Онлайн-анкетирование | 30–50 респ. | Статистика: масштабирование инсайтов |
| **Итого** | | **~50+ респ.** | Полноценная картина |

### Вариант A: Фокус-группы + анкетирование

- 3–4 онлайн ФГ по 5–6 участников (18–24 чел.) + анкета (30–50 чел.)
- ФГ: 90–120 минут, через Zoom / Google Meet
- **Плюсы:** соответствует предпочтению клиента, групповая динамика
- **Минусы:** воронка шире в 2–2,5 раза (рекрутировать 40–50 для 24 участников), сложнее организовать, чувствительные темы раскрываются хуже

### Вариант B: Микс (рекомендуемый)

- **1 фокус-группа** (5–6 чел.) — общие ассоциации, групповая динамика
- **15–20 IDI** (30–45 мин.) — доверие, страхи, конверсионные барьеры
- **Онлайн-анкета** (30–50 чел.) — количественная база
- **Почему рекомендуем:** покрываем оба формата из брифа + количественные данные сверх ожиданий; ФГ — «широкая» картина, IDI — глубина, анкета — статистика; воронка рекрутинга управляемая

### Вариант C: Полный цикл

- **2 фокус-группы** — Tier-1 и Tier-2 отдельно (по 5–6 участников)
- **15–20 IDI** (30–45 мин.), ФГ: 90–120 мин.
- **Расширенное анкетирование** (70–100 респондентов)
- Итого: **~90–130 респондентов**
- Дополнительно: сравнительный анализ Tier-1 vs Tier-2, стратегическая презентация клиенту

### Воронка рекрутинга по вариантам

| | Вариант A | Вариант B | Вариант C |
| :--- | :--- | :--- | :--- |
| Кач. часть: на сессии | 18–24 | 21–26 (1 ФГ + 15–20 IDI) | 25–32 (2 ФГ + 15–20 IDI) |
| Кач. часть: рекрутировать | **40–50** | **30–35** | **35–45** |
| Анкетирование | 30–50 | 30–50 | 70–100 |
| **Общий охват** | **~50–70** | **~50–75** | **~90–130** |

### Разбивка респондентов

- По уровню дохода (нижний / средний сегмент)
- По наличию негативного опыта с кредитными приложениями
- По tier города (Tier-1: Дели, Мумбаи, Бангалор vs Tier-2: Джайпур, Лакхнау, Пуна)

## 4. Пошаговая последовательность работ

Общая схема: **6 фаз, ~5–6 недель.** Фаза 0 (до старта) → Фаза 1 (подготовка, дни 1–5) → Фаза 2 (рекрутинг, недели 1–2) → Фаза 3 (полевые работы, недели 3–4) → Фаза 4 (анализ, неделя 5) → Фаза 5 (сдача, неделя 6).

## 6. Рекрутинг респондентов

### Требования к респонденту

- Живёт в Индии (Tier-1: Дели, Мумбаи, Бангалор или Tier-2: Джайпур, Лакхнау, Пуна)
- Реально пользовался кредитными приложениями (подтверждено, не на словах)
- Доход ₹15 000–50 000/мес.
- Не работает в финтехе
- Рекрутировать с запасом +25–30% (неявка ~20–30%)

### Верификация опыта заимствования

1. Скринер с ловушками (2–3 несуществующих названия кредитных приложений)
2. Конкретные вопросы: сумма, срок, процент
3. Скриншоты кредитного приложения на скрининг-звонке
4. Double screening: короткий звонок перед основным интервью

### Скринер — нужен ASAP

Содержание: демография (возраст, город, доход), подтверждение опыта (названия, суммы, даты), ловушка, исключение финтех-сотрудников, готовность к видеозвонку 30–45 мин.

### Онлайн-анкета (кол. часть)

Развёрнутый опросник по вопросам из раздела 5 брифа (бренд, доверие, ассоциации). Распространяется через WhatsApp / соцсети. Цель: 30–50 ответов. Инструмент: Google Forms / Typeform.
