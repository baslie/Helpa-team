# Дорожная карта проекта: исследование бренда Helpa (Индия)

**Внутренний документ команды**

---

## 1. Суть проекта

Международная финтех-компания выходит в Индию с кредитным приложением **«Helpa»**. Нужно протестировать бренд на реальных индийских заёмщиках. Результат — отчёт с вердиктом: **запускать / доработать / отказаться**.

Методологическая база проекта описана в документе **[«Методы исследования»](Методы_исследования.md)** — mixed-methods дизайн, кросс-культурные фреймворки (Hofstede, GLOBE, Erin Meyer), модели бренда (Keller CBBE, Aaker 5Bs, Brand Linguistics), критерии качества (Lincoln & Guba, Braun & Clarke).

### Что именно тестируем

1. **Восприятие названия** — ассоциации, доверие, запоминаемость, фонетика
2. **Оценка прототипов приложения** — удобство, понятность, ощущение безопасности
3. **Оценка страницы Google Play** — что привлекает, что отталкивает, что побуждает установить
4. **Барьеры и страхи** — связанные с цифровым кредитованием
5. **Сравнение с конкурентами** — RupeeRedee, NIRA, CASHe, mPokket, KreditBee и др.

## 3. Формат исследования

Методологическая база → **[«Методы исследования»](Методы_исследования.md)**.

Клиент в брифе упоминает фокус-группы как предпочтительный формат. Предлагаем **два варианта** + **параллельное анкетирование через чат-бот** для количественной базы.

> **⚡ Клиент выбирает между Вариантами B и C.** Возможен промежуточный вариант.

### Структура: кач + кол

| Компонент | Формат | Количество | Цель |
| :--- | :--- | :--- | :--- |
| **Качественная часть** | IDI + фокус-группы | 20–32 респ. | Глубина: доверие, страхи, ассоциации, барьеры |
| **Количественная часть** | Анкетирование через чат-бот | 50–100 респ. | Статистика: масштабирование инсайтов, процентные срезы |
| **Итого** | | **~70–130 респ.** | Полноценная картина |

Анкетирование через чат-бот (Telegram / WhatsApp) запускается параллельно с интервью. Высокий отклик (знакомая среда для ЦА), визуальные стимулы встроены прямо в поток анкеты.

### Вариант B (рекомендуемый): 1 ФГ + IDI + чат-бот

- **1 фокус-группа** (5–6 чел., 60–90 мин.) — общие ассоциации, групповая динамика
- **15–20 IDI** (глубинные интервью 1-на-1, **40–60 мин.**) — доверие, страхи, конверсионные барьеры
- **Анкетирование через чат-бот** (50–100 чел.) — количественная база
- Итого: **~70–130 респондентов**, онлайн (Google Meet для IDI и ФГ; Telegram / WhatsApp для чат-бота)

### Вариант C (полный цикл)

- **2 фокус-группы** — отдельно для Tier-1 и Tier-2 (по 5–6 участников, 60–90 мин.)
- **15–20 IDI** (глубинные интервью 1-на-1, **40–60 мин.**)
- **Расширенное анкетирование через чат-бот** (70–100 респондентов)
- Итого: **~90–130 респондентов**. Дополнительно: сравнительный анализ Tier-1 vs Tier-2, стратегическая презентация клиенту

### Воронка рекрутинга

| | Вариант B | Вариант C |
| :--- | :--- | :--- |
| Кач. часть: на сессии | 21–26 (1 ФГ + 15–20 IDI) | 25–32 (2 ФГ + 15–20 IDI) |
| Кач. часть: рекрутировать | **30–35** | **35–45** |
| Чат-бот анкетирование | 50–100 | 70–100 |
| **Общий охват** | **~70–130** | **~90–130** |

Набрать заёмщиков в Индии — не проблема. Дипаншу подтвердил: цифровое кредитование в Индии — часть повседневной жизни для нашей ЦА. Рекрутируем с запасом +25–30% (неявка у индийских респондентов ~20–30%).

Основной поток: реклама → скрининг → верификация → расписание → пилот → корректировка гайда. Параллельно: WhatsApp-аутрич и запуск чат-бота.

**Пилот — ключевой чекпоинт.** После 1–2 пробных интервью проверяем: работает ли гайд? Справляется ли Анастасия с акцентом? Нужны ли правки?

**Требования к респонденту:**
- Живёт в Индии (Tier-1: Дели, Мумбаи, Бангалор или Tier-2: Джайпур, Лакхнау, Пуна)
- Реально пользовался кредитными приложениями (подтверждено, не на словах)
- Доход ₹15 000–50 000/мес.
- Не работает в финтехе

**Каналы рекрутинга (по приоритету):**
1. **Facebook / Instagram реклама (лид-формы)** — таргетинг Индия, Tier-1/2, NBFC-приложения. Настраивает Артур
2. **WhatsApp-аутрич** — комьюнити-группы, рассылки
3. **Дипаншу** — прямой рекрутинг через свои связи в Индии
4. **Знакомые команды** — контакты, связанные с Индией / IT
5. **Google Play — майнинг отзывов** — реальные пользователи KreditBee, MoneyTap и др.
6. **LinkedIn** — группы Indian Fintech, Digital Lending India
7. **MyParticipants** — индийская платформа, работала с Blinkit и финпродуктами

## 6. Deliverables — что клиент получит

1. **Аналитический отчёт** — выводы и рекомендации: восприятие бренда, барьеры, точки роста, бенчмаркинг vs конкуренты (по модели Aaker 5Bs)
2. **Чёткая рекомендация** — формализованная: использовать как есть / адаптировать / отклонить — с обоснованием и конкретными шагами по каждому сценарию
3. **Verbatim-цитаты** — ключевые цитаты респондентов с переводом на английский и контекстом (город, доход, пол, метод)
4. **Количественные срезы** — по результатам чат-бот анкетирования (семантический дифференциал, шкалы Лайкерта, непараметрические тесты)
5. **Видеозаписи** — всех интервью и фокус-групп
6. **Карта восприятия бренда** *(опционально)* — визуализация: доверие × дружелюбие × легитимность
7. **Варианты фонетических адаптаций названия** *(опционально)* — только если спонтанно предложены респондентами
